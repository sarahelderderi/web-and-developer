Do you offer ongoing help with our new site once it's launched?

Definitely! The Web and the Developer can provide ongoing support for any website maintenance or enhancements. Take a look at our subscription service.

You can pay for your subscription using a credit card (Visa, Mastercard, Amex) or Paypal. Bank transfer payments are available only for custom invoices or purchase orders.

What are the benefits of a hand-coded website?

When we write code in HTML, CSS and JS, you get to control nearly everything on your website. Because we build your website from the core using a markup language, accessing and modifying the code is easier than with WordPress. This makes it much more flexible when integrating certain additional features. For example, while WordPress uses schema plugins, we can easily employ schema markup on a static HTML website via code.

How much does a website cost?

The age-old question! It completely depends on what you need, but you can rest assured we deliver the best possible value for money and it won’t break the bank. Why not ask for a quote? There’s no obligation and we won’t sulk if you say ‘no’!

What happens to the copyright of my website if I move to another agency?

Your site is yours. You can move if you like, and we will never penalise you for it.

Can I purchase my own domain?

Go ahead. Once you’ve sorted your domain out, we can take things from there. But if you need help, feel free to ask.

Are Web and Developer websites optimised for search engines?

We take a very detailed approach with web page code optimisation. What this means is that we don't just create attractive websites  - we also create websites that have all of the important areas optimised for the search engines ensuring the best start for your website within the search engines.
